# 🌍 今からどこいく？World Travel Game

**「今からどこいく？World Travel」** は、ボタンをクリックするだけで世界のどこかに旅立てる、シンプルで楽しいテキストアドベンチャーゲームです。  
ランダムに選ばれた旅先と同行者が、あなたの1日を彩ります。

---

## ✨ 特徴

- 🌐 世界中の魅力的な場所からランダムに旅先を表示  
- 🎉 ユニークなイベントとお土産アイテム  
- 🧳 個性豊かな3人の同行者のうち1人が旅に同行  
- 🎲 「旅くじを引く」だけのシンプルな操作  
- 📱 モバイルフレンドリーなレスポンシブデザイン  

---
